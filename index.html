Configuração — Google Sheets
Siga estes passos uma única vez. Depois todos acessam o mesmo link e os dados ficam sincronizados.
Passo 1 — Criar a planilha

Acesse sheets.google.com, crie uma planilha nova. O nome não importa.

Passo 2 — Criar o Apps Script

Na planilha aberta: Extensões → Apps Script. Apague o código existente e cole o conteúdo do arquivo apps_script.gs que acompanha este app. Clique em Salvar (Ctrl+S).

Passo 3 — Publicar como Web App

Clique em Implantar → Nova implantação. Tipo: App da Web. Executar como: Eu (seu e-mail). Acesso: Qualquer pessoa. Clique Implantar, autorize as permissões e copie a URL gerada.

Passo 4 — Cole a URL abaixo e salve

A URL tem este formato: https://script.google.com/macros/s/AKfy.../exec

Estas configurações ficam salvas apenas neste navegador.
